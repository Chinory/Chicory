## 🌿 关于那个“抢我名字”的蔬菜

既然你都来了，顺便科普一下这种菜（免得你买错）：

* **它是啥？** 这种菜叫菊苣，因为是在全黑环境下生长的，所以叶子是白色的，尖尖是黄色的。
* **怎么弄？** 切掉根部，剥掉蔫了的叶子。
* **小窍门：** 切开后抹点柠檬汁，不然它会变色（就像我被输入法气变色一样）。
* **什么时候吃？** 1月到3月中旬最肥美。
